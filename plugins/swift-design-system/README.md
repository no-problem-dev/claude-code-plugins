# swift-design-system

Swift Design System を使用した iOS UI 実装のためのスキルプラグイン。デザイントークン、UI コンポーネント、レイアウトパターンのベストプラクティスを提供する。

## インストール

マーケットプレイス経由:

```bash
/plugin marketplace add no-problem-dev/claude-code-plugins
```

## スキル

| スキル | 説明 |
|--------|------|
| `design-system-workflow` | デザイントークン（カラー、タイポグラフィ、スペーシング等）、UI コンポーネント（Button、Card、Chip 等）、レイアウトパターンを活用した SwiftUI 実装ガイド |

## 対象

- SwiftUI ベースの iOS アプリ開発
- no-problem-dev の `swift-design-system` SPM パッケージを使用するプロジェクト
